Prototype Projects

Undergraduate thesis

I have started working on my undergraduate thesis involving Machine Learning specifically Kohonen Self Organising Maps. Stay tuned for more updates.

Simple Sequencing Prototype

I created a web application prototype for Dr.Hussein Suleiman's research group. The application enables instructors to sequence their resources using the IMS Simple Sequencing standard. I implemented it Nodejs and the following technologies: Java, Apache libraries and ffmpeg. I made my flavor of the Apache POI library here. I used various js frameworks for the front end including introjs, guidejs, sequencejs, bootstrap treeview and interactjs. I wrote about it here.

Double Check

Ever got frustrated because someone(or you) hit reply all where they shouldn't have? Have you sent email to the wrong recipient with a similar name or even worse, replied to yourself instead of the other party? I created an outlook addin for a Devpost competition. It enables you to double check the recipients of an email with a single click before sending. The demo can be found here. I wrote it in JavaScript and the source code is available here.

Investec IT Challenge

I took part in Investec IT Explore, a hackathon organised by Investec. In a group a 5, we were required to gamify the youth account application to encourage the culture of saving. We came up with a game that unlocked 360 degrees view of pretty things with more savings.I worked on the backend in Nodejs as well as Reeljs for the front end. We were applauded for the most impressive front end. I wrote a post about it here and the source code can be found here.

Footsteps

I participated in yet another devpost competition that was intended to make people happier. I created an android application called Footsteps. It gives you stories of your role models based on how you are feeling. It was among the top 10 worldwide. I wrote about it here and the source is available here.
